Formuláře

Nemůžeme použít floating a placeholder najednou, musí být buď jedno nebo druhé. Ale stejně musí být v kódu kde je floating placeholder protože jinak to nefunguje floating.

        
            <form >
                <div class="form-floating mb-3">
                    <input type="email" class="form-control" id="floatingEmail" placeholder="" >
                    <label for="floatingEmail">Email</label>
                  </div>
                  <div class="form-floating">
                    <input type="text" class="form-control" id="floatingName" placeholder="" >
                    <label for="floatingName">Jméno</label>
                  </div>
                    <button type="submit" class="btn btn-primary mt-3">Submit</button>
            </form>

Stylizovaný Formulář s Placeholdery

Nemůžeme použít floating a placeholder najednou, musí být buď jedno nebo druhé.

        
        <form>
            <div class="mb-3">
                <label for="email" class="form-label">Email</label>
                <input type="email" class="form-control" id="email" name="email" placeholder="name@example.com" required>
            </div>
            <div class="mb-3">
                <label for="name" class="form-label">Jméno</label>
                <input type="text" class="form-control" id="name" name="name" placeholder="Vaše jméno" required>
            </div>
            <button type="submit" class="btn btn-primary">Odeslat</button>
        </form>

Můj názor na Bootstrap

Tento text obsahuje moje myšlenky o Bootstrapu, které se pravděpodobně ještě vyvinou, jak se s tímto nástrojem seznámím více.

Můj názor na Bootstrap je zatím poměrně nejasný, jelikož ho ještě dobře neznám. Přesto jsem si jistý, že mi pomůže vytvořit hezčí webové stránky a jeho používání není příliš složité. Co mě však trochu zklamalo, byla práce s formuláři. Snažil jsem se vytvořit formulář, ve kterém by fungovaly jak floating, tak placeholder efekty. Bootstrap sice poskytuje elegantní řešení, avšak bohužel se ukázalo, že to nefunguje tak, jak bych očekával. I když existuje kód, který by měl umožnit obě funkce, ve skutečnosti se aktivuje pouze floating efekt. Při hledání řešení jsem si všiml, že ani na oficiální stránce Bootstrapu se placeholder nezobrazoval správně. Když jsem se zeptal ChatGPT, proč tomu tak je, po půl hodině úpravy kódu jsem zjistil, že Bootstrap neumožňuje, aby tyto funkce fungovaly současně.

Důvody, proč se mi Bootstrap líbí:

  • Snadná struktura a použití
  • Stylové komponenty pro rychlé prototypování
  • Kompatibilita s různými prohlížeči

Co mě zklamalo:

  • Problémy s kombinací floating a placeholder funkcí
  • Někdy složité pochopení dokumentace

Dále oceňuji, že mohu použít Bootstrap utility třídy, jako jsou .text-center, .bg-light, .p-3, a .rounded, pro snadné stylování prvků bez nutnosti psát vlastní CSS. To může významně urychlit proces vývoje.

Pokud máte zájem o další informace o práci s Bootstrapem, doporučuji navštívit oficiální stránku Bootstrapu, kde najdete spoustu užitečných zdrojů a příkladů.
